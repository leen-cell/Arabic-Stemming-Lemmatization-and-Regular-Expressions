## Overview

This assignment applies Python and the `re` module to the **United Nations General Assembly Resolutions Corpus (UNCorpus)** — a six-language parallel corpus in Arabic, Chinese, English, French, Russian, and Spanish stored in TMX (Translation Memory eXchange) format.

It is divided into two sections:

| Section | Topic | Points |
|---|---|---|
| A | Regular Expressions | 70 |
| B | Arabic Stemming and Lemmatization | 30 |

---

## Section A — Regular Expressions (70 pts)

Explores the UNCorpus using Python's `re` module.

- **A1** — Investigating the XML structure of the TMX file using regex
- **A2** — Corpus-level statistics: line counts, segment counts per language
- **A3** — Extracting English plain text and computing token/type statistics
- **A4** — Substring matching for the word `"war"` across the English vocabulary
- **A5** — Word frequency histogram and Zipf's Law analysis (top 10, hapax legomena, bar chart)
- **A6** — Top-20 most frequent words per language (EN, AR, ES, RU) with bar charts and linguistic discussion

---

## Section B — Arabic Stemming and Lemmatization (30 pts)

Applies and compares four Arabic NLP tools on the Arabic portion of the UNCorpus.

| Tool | Type |
|---|---|
| NLTK ISRI | Root-based stemmer |
| Tashaphyne | Light stemmer + root extractor |
| Light10 (Larkey et al., 2002) | Affix-stripping light stemmer |
| Alma (SinaTools — SinaLab, BZU) | True lemmatizer + POS tagger |

- **B1** — Vocabulary reduction comparison across all three stemmers
- **B2** — Stemming consistency check on the 50 most frequent Arabic words
- **B3** — Stemmers vs. lemmatizer on 15 UN political/legal Arabic terms
- **B4** — Over-stemming analysis: identifying stems that collapse 5+ unrelated word forms
